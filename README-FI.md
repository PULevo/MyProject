# MyProject – Kevyt tehtävienhallinta SaaS pienille tiimeille

Kevyt SaaS-tyylinen web-sovellus, joka on suunniteltu pienille tiimeille (1–10 käyttäjää) projektien ja tehtävien hallintaan.

Projektia kehitetään full stack -portfolioprojektina käyttäen FastAPI:a, PostgreSQL:ää ja React/Next.js:ää. Tavoitteena on toteuttaa tuotantotasoinen backend-arkkitehtuuri, autentikointi, tietokantarakenne ja skaalautuva SaaS-järjestelmä.

---

# Nykyinen tila

- Projektirakenne luotu

# Projektin tavoite

Rakentaa tuotantotasoinen tehtävienhallintajärjestelmä pienille tiimeille.

Keskeiset tavoitteet:

- Selkeä ja skaalautuva backend-arkkitehtuuri
- Oikea tietokantarakenne ja migraatiot
- Turvallinen autentikointi

- Portfoliokelpoinen koodipohja
